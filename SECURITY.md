# Security Policy

## Supported Versions

| Version     | Supported                                   |
| ----------- | ------------------------------------------- |
| v1          | :white_check_mark: (until December 2023)    |
| < 1.0       | :x:                                         |

## Reporting a Vulnerability

Please report any vulnerabilities to security@umbraco.dk

You can expect to hear back from us within 48 hours.
